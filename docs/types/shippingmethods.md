[<](../index.md) Altapay - PHP Api - Shipping methods
===============================================

Following shipping methods are accepted

```
'LowCost',
'DesignatedByCustomer',
'International',
'Military',
'NextDay',
'Other',
'StorePickup',
'TwoDayService',
'ThreeDayService',
```

Use the one that is closest to your usage, and let the fraud detection be more aquate
