[<](../index.md) Altapay - PHP Api - Languages
==============================================

Following languages are accepted

```
'cs', 'da', 'de', 'en', 'es', 'fi', 'fr', 'ja',
'lt', 'nl', 'no', 'nb', 'nn', 'pl', 'sv','th', 'tr', 'zh',
'et', 'ee', 'it', 'pt', 'eu'
```
where ```nb, nn``` will be converted to ```no``` and ```ee``` to ```et```
